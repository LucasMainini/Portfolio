# Portfólio — Lucas Mainini Pinheiro

Site de portfólio pessoal com apresentação, tecnologias, projetos e formulário de contato via WhatsApp.

## Visão geral
- Navegação por âncoras: Início, Sobre, Projetos e Contato.
- Seção “Sobre mim” com descrição pessoal.
- Seção “Tecnologias” com imagem representativa.
- Cards de projetos com imagem, título e descrição.
- Formulário de contato que abre o WhatsApp com mensagem pré-preenchida.

## Estrutura do projeto
- `index.html`: Página principal do portfólio.
- `styles.css`: Estilos do site.
- `img/`: Imagens usadas nas seções e projetos.

## Como visualizar
Abra o arquivo `index.html` no navegador.

## Personalização rápida
- Atualize textos e títulos no `index.html`.
- Troque as imagens na pasta `img/`.
- Ajuste cores, tipografia e layout em `styles.css`.
- Defina o número do WhatsApp na variável `telefone` do script no final do `index.html`.
